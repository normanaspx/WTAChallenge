# Norman Vicente WTA Challenge


## Architecture
- Kotlin
- MVVM pattern
- LiveData
- dependency injection Hilt 
- Room Database 
- Recyclerview

## Features
- Breaking bad character browser.
- Option to like the character and have it displayed at the top of the list.
